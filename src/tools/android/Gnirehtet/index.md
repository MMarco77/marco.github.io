# Gnirehtet (tethering)

## Usage

- Install

```shell
cd /tmp
wget "https://github.com/Genymobile/gnirehtet/releases/download/v2.3/gnirehtet-rust-linux64-v2.3.zip"
unzip ./gnirehtet-rust-linux64-v2.3.zip
cd gnirehtet-rust-linux64
./gnirehtet install
```

- Run

```shell
$ ./gnirehtet run
```

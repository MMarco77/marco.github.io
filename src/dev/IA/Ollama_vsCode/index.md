# Llama Coder – The Ollama Copilot Integrated with VS Code


## [Ollama – Guide to running LLM models locally](https://www.devturtleblog.com/ollama-guide/)

1. Install Ollama

```shell
$ curl -fsSL https://ollama.com/install.sh | sh
```
 
 2. Test `Ollama`
```shell
$ ollama --help
```
